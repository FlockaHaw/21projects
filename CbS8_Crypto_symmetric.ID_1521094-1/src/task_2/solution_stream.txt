** Алгоритм шифрования: RC4 (Rivest Cipher 4).

** Ошибки реализации:

Ошибка 1: Неправильный модуль в KSA
> Вместо строки j = (j + d[i] + b[i % c]) % 255 Должно быть:
j = (j + d[i] + b[i % c]) % 256

Ошибка 2: Неправильный индекс в PRGA:
> Нужно складывать c[i] + c[j], а не c[i] + c[i].

Ошибка 3: # Было:
> Та же ошибка, но более явно - нужно использовать c[j].
k = c[(c[i] + c[i]) % 256]
> Должно быть:
k = c[(c[i] + c[j]) % 256]
