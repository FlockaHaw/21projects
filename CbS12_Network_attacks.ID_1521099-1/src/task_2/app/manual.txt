Есть возможность добавить сеть или хост в файл ips.txt. Либо можно просто вписать вручную при выполнении программы ( Нужно выбрать 1 пункт при запуске )

Если нет доступных БД в открытой сети, можно запустить локальный докер и он будет сканиться (127.0.0.1:27017):
docker run -d --name mongodb -p 27017:27017 mongo:4.4

Затем нужно запустить venv (source venv/bin/activate)

Запустить scan.py
